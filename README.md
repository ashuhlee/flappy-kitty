
## Flappy Kitty
A simple Flappy Bird clone, but it's a cat 🐾 features randomly generated pipe positions and score tracking. <br>
Made with Lua and Love2D

### Build from Source
#### Requirements
- [Love 2D](https://love2d.org/) version 11 or higher

1. Clone this repository:
```zsh
git clone https://github.com/ashuhlee/flappy-kitty.git
cd flappy-kitty
```
2. Run the app:
```zsh
love .
```


## How to Play
- Any key to start
- Press `Space` to Jump
- Press `Enter` to restart

## Assets Used
Background music - Pecan Pie from [Uppbeat](https://uppbeat.io/t/pecan-pie/boogie)<br>
Sound effects - Pixabay <br>
Pipes - Goodgis
